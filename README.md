# Planni

Planni is a command-line interface (CLI) tool for planning sprints, managing tasks, and achieving your goals one small step at a time. Built with Go, it provides an intuitive way to manage projects and track progress directly from your terminal.

## Features

### Core Features (MVP - Currently working on)
- 📋 Project-based task tracking
- ⏱️ Time management and estimation
- 📅 Sprint planning and management
- 📊 Progress visualization
- 🎯 Goal setting and tracking

### Coming Soon
- 🔄 Sprint retrospectives
- 📈 Progress analytics
- 🔗 Remote storage via SSH
- 📱 Mobile companion app
- 🤝 Team collaboration features
